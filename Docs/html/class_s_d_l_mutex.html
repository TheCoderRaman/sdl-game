Detailed Description

a Mutex class A mutex is used to allow a function to stop progress until some other concurrent progress is finished In short: a mutex prevents concurrent access of data ie. Thread1 locks the mutex, does some stuff. Thread2 meanwhile needs to do things but when it tries to lock the mutex, it must wait untill Thread1 has unlocked and let it continue.
